    SAE3.02 - Développer des applications communicantes

    Tout replier Tout déplier

    Travail à réaliser

    Projet à réaliser individuellement en Python 3.

    OBJECTIF
    Un serveur offre des services de recherche de la présence de mots donnés par des clients dans des documents texte (txt), HTML, PDF et Excel (xslx). Le serveur affiche des informations supplémentaires utiles relatives à ces recherches, par exemple : ligne(s) où le mot a été trouvé, textes autour, etc.

    CONSIGNES
    • Commentaires et noms de variables seront écrits en anglais
    • Un code clair, homogène, simple est attendu. Les programmes sont découpés en fonctions simples. La qualité de votre code est très importante pour l'évaluation. Lire cette synthèse PEP 8 – Style Guide for Python Code pour écrire votre code en la respectant.
    • Ne mettez pas de valeurs constantes (par exemple 12) dans votre code, utilisez des constantes, c'est-à-dire des "variables" dont le nom est écrit en majuscule et dont vous n'allez pas changer les valeurs, par exemple MAX_NB_QUESTIONS = 12, cela permet également de voir à quoi correspondait ce 12).
    • Attention à mettre des chemins relatifs, pas absolus. Exemple : PATH_XLS = "excel"
    • Vous créerez un projet gitlab an ajoutant en tant que developer l'enseignant David Mercier sous le gitlab de l'université d'Artois.
    • Mettez une docstring à vos fonctions. Exemple :
      • def add(a, b):
    """ Return the sum of two arguments """
    return a + b
    • Votre projet contiendra un fichier readme.md écrit en anglais (aide : deepl, reverso, etc.) décrivant ce que fait votre projet et comment l'utiliser (un exemple de projet) avec les dépendances (fichier requirements.txt) (une doc pour requirements.txt). Le fichier readme.md indique principalement comment utiliser votre application.

    • socket serveur (exemple, variables et commentaires en anglais)

      # Server

import socket

# IP Address and port number for the SERVER
SERVER_HOST = "127.0.0.1"	# localhost 
SERVER_PORT = 52300 		# greater than 1024
ENDING_MSG = "q"
  
# 1) Creation of an IPv4/TCP socket:  AF_INET = IPv4 / SOCK_STREAM = TCP
with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
    # 2) Binding the socket
    s.bind((SERVER_HOST, SERVER_PORT)) # Be careful than the paramater is a tuple

    # 3) Waiting for a client 
    print("[Serveur] Serveur prêt, en attente d'un client.") 
    s.listen(1)  # Accept 1 client at most
  
    while True:
        # 4) Establishing the connection with a client
        connection, address = s.accept()
        print(f"[Serveur] Client {address[0]}:{address[1]} vient de se connecter.") 
  
        # 5) Discussing with the client while it sends not ENDING_MSG
        server_msg = f"Vous êtes connecté au serveur. Quel est votre message ?"+\
                f" ({ENDING_MSG} pour quitter)"
        connection.send(server_msg.encode("utf8"))
        client_msg = ""
        while client_msg != ENDING_MSG:
            client_msg = connection.recv(1024).decode("utf8")
            print(f"[Client] {client_msg}")
            server_msg = input("[Serveur] ")
            connection.send(server_msg.encode("utf8"))
        print(f"[Serveur] Client déconnecté. En attente d'un nouveau client.")

    • socket client (exemple, variables et commentaires en anglais)

      # Client
  
import socket

# IP Address and port number for the SERVER
SERVER_HOST = "127.0.0.1"
SERVER_PORT = 52300 
ENDING_MSG = "q"

# 1) Creation of an IPv4/TCP socket:  AF_INET = IPv4 / SOCK_STREAM = TCP
with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
    # 2) Connecting the server
    print(f"[Client] Connexion au Serveur {SERVER_HOST}:{SERVER_PORT}")
    s.connect((SERVER_HOST, SERVER_PORT)) # Watch out the tuple!
    print(f"[Client] Connexion établie avec {SERVER_HOST}:{SERVER_PORT}") 
    
    # 3) Dialogue with the server 
    client_msg = ""
    while client_msg != ENDING_MSG:
        server_msg = s.recv(1024).decode('utf8')
        print(f"[Serveur] {server_msg}")
        client_msg = input("[Client] ")
        s.send(client_msg.encode("utf8"))
    
    # 4) Closing the connection 
    print("[Client] Fin connexion.")
    
    # The with clause automatically closes the connection
