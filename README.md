# Communicating Search Application (SAE3.02)

## Description
[cite_start]This project is a **Client-Server application** developed in **Python 3**[cite: 1, 4]. [cite_start]It allows a client to connect to a server to search for specific keywords across multiple document formats including **TXT, HTML, PDF, and Excel (XLSX)**.

The application supports:
* [cite_start]**Concurrent clients**: Multiple users can search at the same time.
* [cite_start]**Advanced search**: Boolean operators (AND/OR) and Regular Expressions (Regex).
* [cite_start]**Detailed results**: Provides specific locations for findings, such as page numbers (PDF), line numbers (TXT/HTML), or cell coordinates and sheet names (Excel).
